COMBINED DATA - Publish'te her zaman bulunmasi icin gomulur
============================================================

1. Fertigungsplanung_combined.xlsx dosyasini BU KLASORE kopyalayin:
   src/assets/plan/Fertigungsplanung_combined.xlsx

2. Sonra projede build alin:
   npm run build

   Build sirasinda bu Excel bundle icine gomulur (base64). Boylece
   publish/deploy sonrasi combined data her zaman uygulamada bulunur;
   masaustu veya assets URL'sine ihtiyac kalmaz.

3. Test siparisleri (Projekt-Nummer + Produkt) combined datadaki
   siparislerle eslesir; makine atamasi (SL4066 vb.) bu eslesmeye gore yapilir.
